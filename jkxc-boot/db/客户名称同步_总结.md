# 客户名称同步更新 - 完整解决方案总结

## 📋 问题概述

**问题：** 在客户管理中修改客户名称后，订单中心、代账合同等其他模块中的公司名称没有同步更新。

**影响范围：** 9个数据表，包括订单、合同、地址、银行日记账、报销等核心业务数据。

**解决方案：** 使用数据库触发器实现自动同步 + 历史数据修复 + 定期维护检查。

## 💡 技术亮点

### 1. 触发器设计
```sql
-- 智能判断：只在名称变化时执行
IF OLD.corporate_name != NEW.corporate_name THEN
    -- 批量更新关联表
    UPDATE gh_order SET company_name = NEW.corporate_name ...
END IF;
```

### 2. 级联更新
```sql
-- 通过JOIN实现多级关联更新
UPDATE gh_accounting_contract ac
INNER JOIN gh_order o ON ac.order_id = o.id
SET ac.company_name = NEW.corporate_name
WHERE o.company_id = NEW.id;
```

### 3. 幂等性设计
```sql
-- 可重复执行，不会重复更新
WHERE o.company_name != c.corporate_name
```

### 4. 性能优化
- 使用索引加速查询
- 批量更新减少IO
- 条件过滤减少扫描
